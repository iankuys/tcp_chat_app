# Python TCP Chat App

A simple terminal-based TCP chat application built with Pythonâ€™s `socket` library. This app demonstrates core networking concepts like client-server communication, multithreading, and real-time message exchange over TCP sockets.

## Tech Stack

- **Language:** Python 3
- **Networking:** `socket` library
- **Concurrency:** `threading` for handling multiple clients

## Features

- Client-server architecture using TCP
- Supports multiple clients via threading
- Real-time bidirectional messaging
- Clear separation of server and client code
- Lightweight and easy to run locally

## Getting Started

### 1. Run the Server
```bash
python my_tcp_chat.py
```

### 2. Join as client
``` bash
telnet localhost 5555
```
